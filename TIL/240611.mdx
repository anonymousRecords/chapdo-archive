## 📌 뚜레쥬르 관리자 페이지에 로그인 기능 추가
[NextAuth.js](https://next-auth.js.org/)를 사용하여 뚜레쥬르 관리자 페이지에 로그인 기능을 추가하였다.

간단하게만 작업한 내용을 복기하면,
**1. AuthProvider**
```
export default function AuthProvider({ children }: { children: React.ReactNode }) {
  return <SessionProvider>{children}</SessionProvider>;
}
```

SessionProvider는 클라이언트에서 사용할 수 있는 세션을 제공하는 컨텍스트이기 때문에 따로 분리하고, layout 페이지에 SessionProvider가 있는 AuthProvider를 추가하였다.

**2. options & route**
로그인 기능만 필요하기에 `CredentialsProvider`를 사용하였다.
이 options를 route에 전달하여 로그인 페이지를 구현하였다.
