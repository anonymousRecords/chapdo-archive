## 📌 비속어 필터링 기능 구현
```
const [isSlangUsed, setIsSlangUsed] = useState(false);

  const checkSlang = (message: string) => {
    SlangList.some((slang) => {
      if (message.includes(slang)) {
        setIsSlangUsed(true);
        return true;
      }
    });
  };
```

SlangList는 이전 프로젝트에서 만들어 둔 리스트가 있어서 재사용하였다.
이렇게 하면 비속어가 포함되어있는지 확인할 수 있고, 
isSlangUsed가 true이면 경고문이 노출되는 방식으로 구현하였다.

이때 비속어 필터링 기능이 '텍스트', 'cursor chat', 'thread' 이렇게 세 군데에서 사용하는데
중복으로 코드를 작성하였다는 점이다.
초반에 custom hook을 작성하였으나, 잘 진행이 되지 않아서 일단은 이렇게 구현하였다.
